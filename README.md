# House Price Prediction with Geospatial Analysis

## Overview
This project predicts house prices using machine learning techniques combined with geospatial data analysis. The model analyzes various features such as location, property size, number of rooms, and other attributes to estimate the market price of houses.

The project also integrates geospatial visualization to better understand how location influences housing prices.

## Features
- Machine Learning based house price prediction
- Geospatial data analysis
- Interactive data visualization
- Location-based insights for real estate pricing
- Clean and user-friendly interface

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib / Seaborn
- Geospatial analysis libraries
